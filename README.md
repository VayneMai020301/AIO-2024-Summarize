# PI Estimation Calculation
1 . Monte Carlo Method 
$$\pi = \frac{s^2 * N_{c}} {N_{s}}$$

1 . Gregory-Leibniz Series 
$$\pi = 4 * \sum_{i=1}^{n}\frac{(-1)^{i+1}}{2i -1}$$

1 . Nilakantha Series 
$$\pi =  3 + 4 * \sum_{i=0}^{n}\frac{(-1)^i}{(2i + 2)(2i+3)(2i+4)}$$
